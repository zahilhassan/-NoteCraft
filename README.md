# 📝 NoteCraft

A modern and responsive **NoteCraft** application built with **HTML**, **CSS**, and **JavaScript**. Create, organize, edit, and manage your notes with a clean interface, powerful features, and automatic Local Storage support. Designed with a premium glassmorphism UI and a seamless user experience.

---

## ✨ Features

- 📝 Create Notes
- ✏️ Edit Notes
- 🗑️ Delete Notes
- 📌 Pin Important Notes
- ⭐ Mark Notes as Favorites
- 📦 Archive & Restore Notes
- 🔍 Search Notes
- 🏷️ Categories & Tags
- 🎨 Color Labels
- 📄 Rich Text Editor
- 📊 Word & Character Counter
- 💾 Auto Save
- 📂 Import Notes (TXT & JSON)
- 📤 Export Notes (TXT, JSON & PDF)
- 🌙 Dark & Light Mode
- 📱 Fully Responsive Design
- 💾 Local Storage Support
- 🎨 Premium Glassmorphism UI
- ✨ Smooth Animations
- 🔔 Toast Notifications
- ⚙️ Settings Panel
- 📋 Grid & List View
- 🎯 Drag & Drop Note Reordering

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
